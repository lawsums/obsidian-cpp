## 三数之和

给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：** 答案中不可以包含重复的三元组。

## 思路

这题难点有几个地方
1. 两个优化要记得（同时要知道为什么要优化，如果这道题**不需要不重复**，而是记录每个独特的三元组出现了多少次，应该怎么做？）
2. 每次取了一个答案之后如何跳过重复的 j/k 也是重点

## 标准解法
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> ans;
        int n = nums.size();
        sort(nums.begin(), nums.end());

        for (int i = 0; i < n - 2; i++) {
            int x = nums[i];
            if (x > 0) { // 不需要判断了
                break;
            }
            if (i > 0 /* 这里非常重要 */ && x == nums[i - 1]) { // 因为后面的数字是前面的子集，如果x == nums[i - 1]也只能形成一样的三元组，所以跳过
                continue;
            }
            
            int j = i + 1, k = n - 1;
            while (j < k) {
                if (x + nums[j] + nums[k] > 0) {
                    k--;   
                } 
                else if (x + nums[j] + nums[k] < 0) {
                    j++;
                } 
                // sum == 0
                else {
                    ans.push_back({x, nums[j], nums[k]});
                    // 如果后面一个元素和当前元素一样就一直跳过
                    while (j < k && nums[j] == nums[j + 1]) {
                        j++;
                    }
                    while (j < k && nums[k] == nums[k - 1]) {
                        k--;
                    }       
                    j++;
                    k--;             
                }
            }
        }

        return ans;
    }
};

```
