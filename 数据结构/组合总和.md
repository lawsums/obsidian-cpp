## 组合总和


## 思路

这道题的思路就是**逮着一个数可以一直选**, 不像别的题目只能选一次, 所以我们利用**位置来调整它的计算**, 每次从 begin 开始, 代表这个数可以选, 然后可以从 candidates\[begin]选到 candidates\[n - 1], 同时选完了之后继续从 begin 开始 dfs (begin, XXX, XXX), 表示每一次都可以继续选择 candidates\[begin], 就是起到可以**重复选**的作用
退出条件就是**当前数字和大于等于target**, 如果等于就加入答案


## 标准解法

```cpp

class Solution {
private:
    int target;
    vector<int> path;
    vector<vector<int>> ans;

public:
    vector<vector<int>> combinationSum(vector<int> &candidates, int target) {
        this->target = target;
        dfs(0, 0, candidates);
        return ans;
    }

    void dfs(int begin, int sum, vector<int> &candidates) {
        if (sum >= target) {
            if (sum == target) ans.push_back(path);
            return;
        }

        for (int i = begin; i < candidates.size(); i++) {
            path.push_back(candidates[i]);
            dfs(i, sum + candidates[i], candidates);
            path.pop_back();
        }
    }
};

```
