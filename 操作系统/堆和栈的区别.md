
| 维度   | 栈                     | 堆                            |
| ---- | --------------------- | ---------------------------- |
| 管理方式 | 编译器**自动分配/释放**        | **手动**使用   new 分配、 delete 释放 |
| 内存结构 | ==先进先出==，类似栈（Stack）结构 | ==无序结构==，需要通过指针访问            |
| 大小限制 | **固定**（操作系统预设，如几 MB）  | 动态（可拓展，**受物理内存限制**）          |
| 分配效率 | 高（仅需移动**栈指针**）        | 低（==可能产生内存碎片==）              |
