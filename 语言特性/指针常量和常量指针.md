指针常量和常量指针是C/C++中容易混淆的概念，核心区别在于"常量"修饰的对象不同：

1. **常量指针（pointer to const）**：
   - 本质是指针，指向一个"常量"（不能通过该指针修改所指内容）
   - 声明形式：`const int* p` 或 `int const* p`
   - 特点：指针指向的值不可修改，但指针本身可以指向其他地址

2. **指针常量（const pointer）**：
   - 本质是常量，这个常量是指针类型（指针本身的值不可修改）
   - 声明形式：`int* const p`
   - 特点：指针本身的指向不可改变，但可以通过指针修改所指对象的值

记忆技巧：看"const"靠近谁，靠近数据类型（如int）则是常量指针，靠近指针符号（*）则是指针常量。

以下是指针常量和常量指针的代码示例，帮助理解两者的区别：

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;
    
    // 1. 常量指针（pointer to const）
    // 指向常量的指针，不能通过该指针修改所指内容
    const int* p1 = &a;
    // *p1 = 30;  // 错误！不能通过p1修改a的值
    p1 = &b;       // 正确：指针本身可以指向其他地址
    
    // 2. 指针常量（const pointer）
    // 指针本身是常量，不能改变指向，但可以修改所指内容
    int* const p2 = &a;
    *p2 = 30;      // 正确：可以通过p2修改a的值
    // p2 = &b;    // 错误！不能改变p2的指向
    
    // 3. 指向常量的指针常量（结合体）
    const int* const p3 = &a;
    // *p3 = 40;   // 错误：不能修改所指内容
    // p3 = &b;    // 错误：不能改变指向
    
    cout << "a = " << a << ", b = " << b << endl;
    return 0;
}
```

总结：
- 常量指针（`const int* p`）：保护数据不被修改，但指针可以"搬家"
- 指针常量（`int* const p`）：指针不能"搬家"，但可以修改所指数据
- 两者组合（`const int* const p`）：既不能修改数据，也不能改变指向
