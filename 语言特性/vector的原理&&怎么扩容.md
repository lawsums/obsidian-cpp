Vector 是一种动态数组容器，其核心原理是通过一篇**连续**的内存存储元素，同时维护着指向内存的起始地址的指针（`begin`）、当前元素数量（` size `）和容量（` capacity `）

扩容机制如下：
1. 当容器满了的时候，我们需要分配一篇更大的内存，一般是**两倍**原先的内存
2. 然后将元素迁移到新的内存中
3. 建立一个新的指针`tmp`指向原先的 `vector`，将原先的 ` vector ` 指针指向新的 ` vector ` 
4. 释放原先的 `vector` 的内存，删除 `tmp`

这种扩容策略保证了`vector`的随机访问效率（时间复杂度O(1)），但扩容过程本身是耗时的（时间复杂度O(n)），因此提前通过`reserve(n)`预留足够容量可以避免频繁扩容带来的性能损耗。
