总体的框架是调用发送 PDU请求实现各个功能

| 功能     | TcpClient/Book/Opewidget                                    | TcpServer |
| ------ | ----------------------------------------------------------- | --------- |
| 进入文件夹  | Book: 将文件夹条目**doubleclicked** 信号和 enterDir 函数联系在一起          |           |
| 返回上一级  | Book: 将**返回按钮** click 信号和 [[returnPreDir]] 函数联系在一起          |           |
| 创建文件夹  | Book: **创建文件夹** click 信号->createDir                         |           |
| 上传文件头  | Book: **上传文件** click 信号->[[uploadFile]]                     |           |
| 上传文件数据 | Book: m_pTimer **timeout** 信号 (设置为 1 秒)->[[uploadFileData]] |           |
| 下载文件数据 | Book: **下载文件** clicked 信号->downloadFile                     |           |
|        |                                                             |           |
